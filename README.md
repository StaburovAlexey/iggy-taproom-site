## 🎯 О проекте

iggy-taproom — интерактивная сцена бара на Three.js с навигацией, пиксельным PS1-стилем и интерфейсом меню, афиши и музыки.

## 🧩 Что сделано

- 3D сцена бара с множественными источниками света, тенями и атмосферной пылью.
- PS1-стилизация: low-poly упрощение, квантизация и dithering в шейдерах, пиксельная фильтрация текстур.
- Камера по точкам интереса с плавными перелётами на GSAP.
- Интерактивная навигация по разделам: меню, мероприятия, музыка.
- Всплывающие окна меню, афиши и музыкального плейлиста на Web Components с Shadow DOM.
- Данные меню (пиво/еда/закуски) с описаниями и ценами.
- Аудио-плейлист с выбором трека и обработкой автозапуска после жеста пользователя.
- Прелоадер с прогрессом загрузки моделей и текстур.
- Анимированные персонажи и объектные модели для hover-подсказок категорий.
- Адаптивный режим: компактная раскладка UI, пересчёт FOV и масштаба рендера.

## ✅ Особенности проекта

- Яркий и цельный визуальный стиль с ретро-атмосферой.
- Полностью кастомные 3д модели сделанные в Blender.
- Плавная навигация и понятный UX для просмотра меню и мероприятий.
- Модульная архитектура и разделение логики по файлам.
- Web Components с Shadow DOM обеспечивают изоляцию UI.
- Легко масштабировать контент: меню, музыка и афиша обновляются данными.
- Наложение текстур сделано в three.js, что бы снизить вес моделей.
